# menu-C#-console

Exercício de menu simples com 6 funções. Feito em C# no console. O usuário pode utilizar o programa novamente ao término de uma função.



Temos as seguintes funções:

1 - Cálculo do IMC

2 - Mostrar número Maior e Menor

3 - Par ou Ímpar

4 - Equação do 2º grau

5 - Fahrenheit para Celsius

6 - Celsius para Fahrenheit



Os valores inseridos pelo usuário são validados conforme a necessidade. Exemplo: no cálculo de IMC o peso e a altura não podem ser negativos. Nas funções de conversão de temperatura os valores inseridos não podem estar abaixo do zero absoluto.
